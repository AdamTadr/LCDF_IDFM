🚂 Le Compagnon du Ferrovipathe

Application mobile regroupant l'histoire ferroviaire et les données techniques des matériels roulants. 
L'accent est mis sur une navigation fluide et une consultation multimédia simplifiée.

🚀 Fonctionnalités

Encyclopédie du rail : Histoire détaillée des lignes et des modèles de trains. 
Données techniques : Fiches complètes sur les spécificités du matériel. 
UI Moderne : Affichage dynamique via des carrousels de sélection. Redirections natives :Lecture de vidéos et photos via l'application YouTube. 
Contact direct par Mail : ouverture automatique du client mail (Gmail) avec destinataire et message pré-remplis pour les suggestions. 🛠

Stack Technique
Langage : Dart (Flutter). 
Logiciels : Visual Studio Code.
